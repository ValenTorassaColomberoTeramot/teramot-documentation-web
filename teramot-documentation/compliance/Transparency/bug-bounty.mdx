# 🛡️ Bug Bounty Program

At Teramot, we value and respect the work of security researchers. If you have discovered a vulnerability or security issue in any of our systems, we encourage you to participate in our **Bug Bounty Program**.

Please send your report to:

👉 **[support@teramot.com](mailto:support@teramot.com)**

Make sure to include enough details to help us reproduce and validate the issue. Responsible disclosure is appreciated.
